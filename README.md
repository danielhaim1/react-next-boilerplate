# react-next-boilerplate

a React + Next.js template, prepackaged w/ SASS, Autoprefixer, PostCSS, and cssnano

## Installation

```shell
$ npm install
```

## Commands

```shell
$ npm run dev
$ npm run start
$ npm run build
```
